# PlanEvaluation Maintainer Documentation

This documentation set is intended for medical physicists and technical maintainers who need to keep the package operational, validate updates, and safely modify report logic.

## Documents

1. 01_Architecture.md
   System architecture, module responsibilities, and runtime flow.

2. 02_Report_Definitions_and_Config.md
   XML report definition format, configuration entries, and matching behavior.

3. 03_Maintenance_Workflow.md
   Change process, validation checklist, and recommended release steps.

4. 04_Known_Risks_and_Technical_Debt.md
   Current technical concerns and practical mitigation strategy.

## Scope

These documents describe the main package in the workspace root. The For Distribution folder contains a copied runtime package and should be kept synchronized intentionally during release packaging.
